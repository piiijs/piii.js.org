Configurações

:house:

/**
 * Sintaxe.
 */
const piii = new Piii({
  filters: {Array},
  aliases: {Object},
  repeated: {Boolean},
  censor: {String|Function},
  cleaner: {Function}
});

(Clique nos links abaixo para saber mais.)

  • filters ― Lista de filtros (não há nada definido por padrão).
  • aliases ― Alfabeto alternativo para alguns caracteres.
  • repeated ― Informa se deve ignorar letras repetidas.
  • censor ― Valor que substituirá os palavrões dentro da string.
  • cleaner ― Removedor de acentos (usa diacritics por padrão).

O que são palavrões?

Piii.js

Um avançado filtro de palavrões.

GitHub Documentação Download

Características

Você poderá:

  • Ignorar se as letras estão em maiúsculas ou minúsculas.
  • Ignorar o uso de qualquer tipo de acentuação.
  • Gerênciar um alfabeto personalizado (veja sobre leet).
  • Ignorar letras repetidas.
  • Criar um filtro personalizado.
  • Manipular o palavrão antes de filtrá-lo.

Instale-o com npm:

npm install --save piii

...Ou veja aqui outras formas de instalar.

Piii.js

Um avançado filtro de palavrões.

DOCUMENTAÇÃO GITHUB

Características

Ignorar se as letras estão em maiúsculas ou minúsculas.

Ignorar o uso de qualquer tipo de acentuação.

Gerênciar um alfabeto personalizado.

Ignorar letras repetidas.

Criar um filtro personalizado.

Manipular o palavrão antes de filtrá-lo.

Primeiros Passos

Instale-o com npm:

npm install --save piii

E a instale a lista de filtros a serem usados:

npm install --save piii-filters

Exemplo de uso:

"use strict";

const Piii = require("piii");
const piiiFilters = require("piii-filters");

const piii = new Piii({
  filters: [
    ...Object.values(piiiFilters)
  ]
});

console.log(piii.has("Foda-se!"));
// true

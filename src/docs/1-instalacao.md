title: "Instalação"
---

Instale-o com *npm*:

```bash
npm install --save piii
```

Ou com *bower*:

```bash
bower install --save piii.js
```

Ou importe-o de [*unpkg.com*](https://unpkg.com):

```html
<script src="https://unpkg.com/piii"></script>
```

Ou baixe-o [aqui](https://git.io/vNx9H).
